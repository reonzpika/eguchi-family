# UI/UX Designer Skill

Responsive-first, self-improving UI/UX skill for this project.

## Attach in Cursor

- **Skills**: In Cursor settings or project rules, add the skill path: `.cursor/skills/ui-design` or `${PROJECT_ROOT}/.cursor/skills/ui-design`.
- **Claude**: In `.claude/settings.json`, set `"skills": ["${PROJECT_ROOT}/.cursor/skills/ui-design"]`.

## Contents

- **SKILL.md** – Main instructions (principles, workflow, templates).
- **anti-patterns.md** – Banned patterns; updated from feedback.
- **design-system.md** – Evolving tokens and components; prefer `docs/design-system.md` for full project system.
- **feedback/** – Per-session feedback (timestamped). Move old files to `feedback/archive/` after 14 days.
